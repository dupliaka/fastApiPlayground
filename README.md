# Fast API Playground

Upload image to unicorn server

### To run locally

install uvicorn
```commandline
pip install uvicorn
```

install fast api

```commandline
pip install fastapi
```

run uvicorn server 

```commandline
uvicorn main:app --reload
```

## Run tests 

install pytest

```commandline
pip install pytest
```

launch tests
```commandline
pytest main.py
```
